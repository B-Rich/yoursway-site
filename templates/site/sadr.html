Static Analysis Done Right

the alien technology
 

Powerful static analysis done in weeks will be a reality. Soon.

And the reality will be open-source.

SADR is designed from ground up to power our own multi-language IDE and to adhere to our high quality standards.

We're currently working on a building blocks support. SADR will allow you to reuse existing parts of the analysis implementations by composing your engine as a set of building blocks. You won't ever reimplement local variables or function closures again.

Examples of the building blocks are:

  • local variables (Ruby, PHP)
  • global variables
  • scoped variable (Python, JavaScript, Pascal, C)
  • first-class functions (Python, JavaScript)
  • message passing (Ruby, Objective C, Small Talk)

Other cool features already working or planned to be implemented soon are:

  • automatic incremental analysis (no work needed from language implementor’s part)
  • on-demand + locally iterative analysis with automatic hot switching
  • infrastructure for adding explicit custom heuristic decision-making algorithms
  • smart design — forget casts and switches
