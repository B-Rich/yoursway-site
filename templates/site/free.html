Creating fancy Mac OS X .dmg's

YourSway create-dmg builds a .dmg for you with a custom image background, specified icon positions and set to open in the Icon View. This tool requires a Mac — unfortunately, no solution exists to create .dmg files on non-Mac platforms.

This tool is licensed under GNU General Public License, because it incorporates portions of GPL-licensed code from Adium repository. You can modify this program to suite your needs, but you have to provide the source code of any modifications upon request.

create-dmg 1.0 is available currently. That's a stable release, and is confirmed to work with Mac OS X Leopard.

Running Eclipse and OSGi on a Mac

YourSway Eclipse OS X Repackager turns a regular Eclipse installation (or your RCP application, or your OSGi-based application) into a proper Mac OS X bundle which looks like a single file in Finder. Get the familiar experience, forget the multifile nightmare.

YourSway Eclipse OS X Repackager 1.2 is available currently. That's a stable release, and is confirmed to work with Eclipse 3.4-based applications.

Continuous Builds

“Release early, release often” is not that easy once you realize you have to bring up a continuous builds system. And if you're starting new projects often, the maintenance of the build system takes quite some time.

Enter YourSway Builder. We created this tool with a single idea in mind: create a hosted solution for continuous builds, which would be as easy to maintain as hosted bug trackers and hosted source control systems are.

YourSway Builder runs on Google App Engine. We don't currently provide accounts on our installation, but you can sign up for your own Google App Engine account and upload our application in less than five minutes. That's it — no server management, no Apache configuration, no more broken email notifications or unavailable build systems.

You also need to deploy our client application on one or more builder machines. There's no reconfiguration or redeployment required to change the build process or even set up builds of a new project. The client is completely controlled from the Web application and even updates itself automatically, so you can forget about it once it's set up. (Estimated client deployment time: 5 minutes on a Mac, 20 minutes on Linux.)

Everything else is controlled by a simple declarative script you upload into the running Web application.

YourSway Builder is currently being developed. Milestone 0.0 is finished, milestone 0.1 is almost finished (see the bug tracker). That's alpha quality, but we're actively using it here at YourSway.

Automatic Updates

YourSway Software Update for Java. Once you get to distributing your application publicly (or enterprise-wide), you have to think of a way for it to stay up-to-date while causing the least trouble to your beloved users. YourSway Software Update allows you to quickly add a reliable and powerful cross-platform update functionality to your application.

YourSway Software Update works with both plain Java applications and OSGi-based applications. In particular, it is compatible with Eclipse and Rich Client Platform apps.

We aimed to make a very simple user interface, but still pack in a few advanced features. YourSway Software Update gracefully handles a ton of exceptional conditions. You get a completely modeless UI by default, and you can replace it with a custom one if you like.

YourSway Software Update is licensed under Apache License 2.0, which means it is free for any use, you can distribute and/or modify it.

YourSway Software Update SDK is under active development at the moment. We are going to start using it in our projects in October'08.
