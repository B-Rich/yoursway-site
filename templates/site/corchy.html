Corchy

To-do lists in sync with Basecamp and your bug trackers

 

This project is on a very early stage of development. We’re sharing a downloadable binary and a screencast to demonstrate the overall idea. Please try it and share your opinion.

If you would be interested to get the source code and hack on it, please contact us. (We’re not yet sure if we want this to be an open-source, free or commercial tool)

 

Dead simple: “Project:”, “- task”, “@tag”

Syncs with Basecamp (and, in the future, many other bug trackers / web apps)

Version control-friendly: keeps To-Dos in a plain text file

Tagging: associate arbitrary metadata with your tasks
