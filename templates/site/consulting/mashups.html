The story is no longer about the Web or the desktop alone. If you need a desktop application, we can bet you want it to interact with web services. If you need a Web application, chances are you also want a matching desktop widget or a full-featured client for offline use, or maybe a matching iPhone app. We recognized that to be strong in any of these areas, we have to be strong in all of them. That’s why we have both Web, Win32 and Mac OS X people on staff, working as a team to bring you integrated solutions.

Development

Here’s what we can do for you:

  • Cross-platform desktop applications.
    We use Java to minimize development costs and complexity, but we employ many advanced techniques to provide native experience on both platforms.
  • Platform-native applications.
    We have experience developing Mac OS X applications in Cocoa and Windows applications on Win32 API or ATL/WTL. However, in our opinion, using Java is the best approach in many ways. While in some cases making a native Mac OS X application is reasonable, we strongly recommend against doing native development for Windows.
  • Web development. We employ Python and Ruby to create stunning web applications faster and easier. Pylons, Django, Rails, Merb or Ramaze, plus unobtrusive JavaScript with jquery or prototype.
  • We can design your application to be hosted on Amazon S3 or Google App Engine.

An advanced mashup: Taskus

We have developed a solution which helps developers to employ personal planning and stay on top of their tasks spread over the various bug trackers and project management software.

Taskus is a desktop application which allows you to keep all your tasks in a simple text file, with 2-way synchronization with several bug tracking and project management applications (with more to come). Taskus is available for both Windows and Mac OS X.

myTaskus is a Web portal which is kept in sync with desktop Taskus and allows you to view and edit your tasks while your personal computer is not available.

iTaskus is a soon-to-be-released iPhone application which is kept in sync with myTaskus and also allows you to view and edit you tasks on the go.
